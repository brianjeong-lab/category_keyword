# kb-daas-mvp-dataflow
GCP DataFlow 처리를 위한 소스

## 테스트 방법
1. 가상환경
<PRE>
  $ virtualenv data-sci-env -p python3
  $ source data-sci-env/bin/activate
</PRE>
2. 설치
<PRE>
  $ pip install timezonefinder pytz
  $ pip install apache-beam[gcp]
</PRE>
